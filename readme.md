Create an ephemeral API token for use in client-side applications with the OpenAI Realtime API. 

## Requirements

- OpenAI API key

## Installation / Usage

```bash
% npm install
```

Start the service

```bash
% OPENAI_API_KEY=<api-key> npm start
```
